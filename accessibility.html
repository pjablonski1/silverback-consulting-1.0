Accessibility Statement

Last updated: June 23, 2026

Our Commitment

Silverback Consulting LLC is committed to making its website and client portal usable for as many people as possible, including visitors who use assistive technologies.

Current Accessibility Measures

The website uses responsive layouts, semantic headings, visible focus states, accessible form labels, keyboard-friendly links and buttons, readable color contrast, skip-link support on the main page, and reduced-motion handling where supported.

Ongoing Improvements

Before public launch, Silverback should test the site and portal with keyboard navigation, screen reader tools, mobile devices, browser zoom, and contrast checks. Any third-party tools such as DocuSign, payment processors, Plaid, or authentication providers should also be reviewed for accessibility.

Feedback

If you experience difficulty using any part of the website or portal, contact Silverback Consulting LLC at sbph2026@gmail.com. Please include the page, device, browser, and a description of the issue so the team can review it.
